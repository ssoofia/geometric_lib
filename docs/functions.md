```python
    import math # Подключаем библиотеку math для использования pi в вычислениях
```
####  Функция для нахождения площади круга
```python
def area(r):

    # Принимает число r (радиус), возвращает площадь круга

    return math.pi * r * r
```
> **Параметры:** _r (float): радиус круга_
>
>**Возвращаемое значение:** _area (float): площадь круга_
>
>**Пример вызова:** _area(5)  78.53981633974483_
---
####  Функция для нахождения периметра круга
```python 
def perimeter(r):

    # Принимает число r (радиус), возвращает периметр круга

    return 2 * math.pi * r
```

> **Параметры:** _r (float): радиус круга_
>
>**Возвращаемое значение:** _perimeter (float): периметр круга_
>
>**Пример вызова:** _perimeter(5)  31.41592653589793_

# rectangle.py 


####  Функция для нахождения площади прямоугольника
```python
def area(a, b):

    # Принимает числа a (длина) и b (ширина), возвращает площадь прямоугольника

    return a * b
```
> **Параметры:** _a (float): длина прямоугольника, b (float): ширина прямоугольника_
>
>**Возвращаемое значение:** _area (float): площадь прямоугольника_
>
>**Пример вызова:** _area(4, 5) 20.0_
---
####  Функция для нахождения периметра прямоугольника
```python 
def perimeter(a, b):

    # Принимает числа a (длина) и b (ширина), возвращает периметр прямоугольника

    return 2 * (a + b)
```

> **Параметры:** _a (float): длина прямоугольника, b (float): ширина прямоугольника_
>
>**Возвращаемое значение:** _perimeter (float): периметр прямоугольника_
>
>**Пример вызова:** _perimeter(4, 5) 18.0_

# triangle.py 



####  Функция для нахождения площади треугольника
```python
def area(a, h): 

    # Принимает числа a (длина стороны) и h (высота, провенная к этой стороне), возвращает площадь треугольника

    return 1/2 * a * h 
```
> **Параметры:** _a (float): длина стороны треугольника, h (float): высота, провенная к этой стороне_
>
>**Возвращаемое значение:** _area (float): площадь треугольника_
>
>**Пример вызова:** _area(4, 5) 10.0_
---
####  Функция для нахождения периметра треугольника
```python 
def perimeter(a, b, c):

    #  Принимает числа a, b, c (стороны треугольника), возвращает периметр треугольника

    return a + b + c 
```

> **Параметры:** _a (float): первая сторона, b (float): вторая сторона, c (float): третья сторона_
>
>**Возвращаемое значение:** _perimeter (float): периметр треугольника_
>
>**Пример вызова:** _perimeter(3, 4, 5) 12.0_
